## Overview
Human picks which of the Liquid Glass tab bar polish variants built in convert-tabbar-b looks best, because 'make it look nicer' is a subjective taste call with no objective acceptance bar.

## Do
Run the app, flip through each value of the `tabBarGlassVariant` constant in Sources/DesignSystem/LiquidGlassTabBar.swift on the simulator, and choose the preferred variant. Record the chosen variant (and any tweak requests) in the worklog so the follow-up hardcode task can implement exactly it.

## Done when
A human has viewed the variants on the simulator and recorded which variant to keep (plus any requested adjustments) for the convert-tabbar-d hardcode task.
