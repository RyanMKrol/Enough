## Overview
A 'Start my review' App Intent that opens the app straight into a review session, discoverable from Siri, Shortcuts, and Spotlight. No separate extension target — the intent lives in the main app.

## Do
Add an AppIntent (e.g. StartReviewIntent) with a clear title and openAppWhenRun = true; in perform(), route via the shared hook from reviews-surface-route (AppState.requestStartReview()) on the main actor, then return .result(). Register it with an AppShortcutsProvider exposing a natural-language phrase (e.g. 'Start my review in Enough') so it surfaces in Shortcuts/Spotlight. Ensure it compiles under the app target's Swift 6 / MainActor isolation.

## Done when
Building the app registers StartReviewIntent and its AppShortcut so it appears in the Shortcuts app; running the intent opens the app into the Reviews tab and starts a review session through the existing StudyService path; a unit test asserts the intent's perform wiring calls the shared start-review hook; xcodebuild test/build stays green.
