Create `Sources/Services/ReadinessCalculator.swift`:
```swift
enum ReadinessCalculator {
  static func readiness(
    tripScenarios: [ScenarioInfo],
    decks: [DeckInfo],
    progress: (String) -> DeckProgressService.DeckProgress?
  ) -> (percent: Int, line: String)
}
```
The required types already exist and build: `ScenarioInfo` / `DeckInfo` in Sources/ContentKit/Models.swift (DeckInfo has `.scenario` and `.cardCount`), and `DeckProgressService.DeckProgress` in Sources/Services/DeckProgressService.swift (has `.learned`). Pin these rules EXACTLY:
- Only decks whose `scenario` is in `tripScenarios` participate; a trip scenario with zero matching decks is ignored entirely (affects neither percent nor line).
- `percent` = Sum(learned) / Sum(cardCount) * 100 over participating decks, rounded half-away-from-zero via `.rounded()`, clamped 0...100; nil progress => learned 0; empty denominator => 0.
- Per-scenario coverage = Sum(learned)/Sum(cardCount) over that scenario's decks; a scenario is HANDLED when coverage >= 0.6.
- Casual-noun map: eating-out->food, greetings->greetings, getting-around->directions, nightlife->nightlife, shopping->shopping, emergencies->emergencies; unknown id -> its `title` lowercased.
- Line: no handled scenarios (incl. nothing learned) => "Start with <first trip scenario's noun>." (noun capitalised at sentence start, e.g. "Start with food."). All participating scenarios handled => "You're covered. Keep it fresh.". Otherwise => "You can handle <list>. <Next>'s next." where <list> joins handled nouns in trip order (one: "food"; two: "food & greetings"; 3+: "food, greetings & directions") and <Next> is the LOWEST-coverage unhandled scenario's noun with first letter capitalised (ties -> first in trip order).
- The calculator is PURE: no `import SwiftData`, no `import SwiftUI`. Keep the return tuple signature verbatim so T054 (Progress tab) consumes it unchanged.

Add `Tests/ReadinessCalculatorTests.swift` (XCTest) building fixtures by constructing `ScenarioInfo`/`DeckInfo` values directly and returning `DeckProgress` from a dictionary-backed closure.
