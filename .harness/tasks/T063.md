## Overview

🔒 needs-human. App icon + launch assets — a brand/taste deliverable the owner picks or
commissions; agents shouldn't invent the brand mark unsupervised.

## Do

1. Produce a 1024×1024 app icon (suggested direction, from the design language: the three-card
   fanned stack, or the "Enough." wordmark period, on white — quietly joyful, no gradients-for-
   gradients'-sake). Tools: any — export PNG without alpha.
2. Add it to an asset catalog (e.g. `Sources/Assets.xcassets/AppIcon.appiconset`) and reference
   it from `project.yml` (`ASSETCATALOG_COMPILER_APPICON_NAME: AppIcon`).
3. Optionally set a launch screen background color matching canvas `#f2f2f7` via Info.plist
   `UILaunchScreen` keys.
4. Verify the icon shows on the simulator home screen after `./build_run.sh`.

## Done when

The icon renders on the simulator home screen, CI stays green, and the owner marks this done via
`.harness/scripts/mark-done.sh T063`.
