# T008 — Streak pill with flickering flame

## Overview

Add the streak pill shown top-right on the Learn tab (and reused by Progress): a white pill with
a softly flickering flame icon and a bold day count. Design values from design-spec.md §1.4
(flame flicker) and §4.

## Do

1. Create `Sources/DesignSystem/StreakPill.swift` with exactly this view:
   - `struct StreakPill: View` with `init(count: Int)`.
   - Body: an `HStack(spacing: 5)` inside a `Capsule`:
     - Flame: `Image(systemName: "flame.fill")` in `EnoughColor.streakAmber` (`#de6907`),
       ~15pt. It flickers continuously: scale oscillates 1.0 ↔ 1.08 and rotation −3° ↔ +3°,
       `.easeInOut(duration: Motion.flicker)` (2.2s) repeating forever with autoreverse —
       subtle, per the design's "small, physical, purposeful".
     - Count: `Text("\(count)")` 15pt bold, `EnoughColor.label`.
   - Pill chrome: fill `EnoughColor.surface` (white), padding 7 vertical / 12 horizontal,
     resting-card shadow `Color.black.opacity(0.05)`, radius 3, y 1.
2. Add a `#Preview` showing `StreakPill(count: 6)` on a `EnoughColor.canvas` background.

Conventions: single app target, SwiftUI only, 2-space indent, SwiftLint strict (≤130 cols) +
swift-format strict. Display-only — the count is computed elsewhere (StatsService, a later
task). Touch only `Sources/DesignSystem/**` (and `Tests/**` if adding optional tests).

## Done when

- `Sources/DesignSystem/StreakPill.swift` compiles with the exact name and `init(count: Int)`
  signature (Home and Progress tasks embed it verbatim).
- The flame is `flame.fill` in `#de6907` with the continuous 2.2s scale+rotate flicker; the
  count renders bold beside it; the pill is white with the subtle resting shadow.
- The `#Preview` block exists as described.
