# T044 — Home screen assembly (header, day counter, sections, navigation)

## Overview

Assemble the Learn tab (Home) from the T043 components: header with trip subtitle + streak
pill, Continue hero, reviews banner, and the "Your decks" list — backed by a view model that
reads real progress/stats, and wired into the main shell's Learn tab in place of its
placeholder.

## Do

Design reference: .harness/docs/designs/design-spec.md (read-only reference) §4. Architecture:
.harness/docs/designs/architecture.md (read-only reference) (AppServices via `\.services`).

1. `Sources/Features/Home/HomeViewModel.swift` — `@Observable final class HomeViewModel`,
   `init(services: AppServices)`, with a `func reload()` that recomputes (call it from
   `HomeView.onAppear` so the ring re-animates and data refreshes on every visit):
   - `subtitle: String` = "\(durationLabel) in \(destinationLabel) · Day \(n)".
     durationLabel: trip duration "weekend" → "Weekend", "week" → "A week".
     destinationLabel — PINNED mapping: countryId "japan" → "Tokyo", "france" → "Paris",
     "germany" → "Berlin"; any other id falls back to the catalog country `name`.
     `n` = `TripStore.dayNumber(now: services.dateProvider.now)` (day 1 = start date).
   - `streak: Int` from `StatsService.currentStreak()`.
   - `continueDeck: (deck: DeckInfo, progress: DeckProgressService.DeckProgress)?` — the first
     OWNED deck (catalog order, active country only) with `0 < learned < total`; if none, the
     first owned deck with `learned == 0`; if none, nil (hide the card).
   - `continueDetailLine: String` — "\(learned) of \(total) cards · about \(m) min left" where
     `m` = remaining cards × 25 seconds, rounded to whole minutes, minimum 1.
   - `dueCount: Int` from `DeckProgressService.totalDue()` — banner shows only when > 0.
   - `deckRows: [(deck: DeckInfo, progress: Double, status: DeckRowStatus)]` — owned decks of
     the active country; status: learned == total → `.learned`; learned == 0 → `.new`; else
     `.progress(learned, total)`.
2. `Sources/Features/Home/HomeDestination.swift` —
   `enum HomeDestination: Hashable { case deckDetail(deckId: String) }`.
3. `Sources/Features/Home/HomeView.swift` — `struct HomeView: View`. A `NavigationStack` over a
   ScrollView on `EnoughColor.canvas` (`#f2f2f7`), horizontal padding 22, section gaps 18:
   - Header: HStack — VStack(leading): "Learn" 34 Bold (tracking −0.02em) + subtitle 15
     Regular `#6b6b70`; Spacer; `StreakPill(count: vm.streak)`.
   - `ContinueCard` (only when `continueDeck != nil`) with progress = learned/total; `onPlay`
     is an empty closure with `// TODO(T047): launch learn session`.
   - `ReviewsBanner(dueCount:)` (only when dueCount > 0); `onReview` empty with
     `// TODO(T049): launch review session`.
   - `EyebrowLabel("Your decks")`, then a white card (radius 20) stacking `DeckListRow`s with
     hairline dividers (`EnoughColor.hairline`) inset to align after the 34pt icon (leading
     inset 46). Each row wrapped in `NavigationLink(value: HomeDestination.deckDetail(deckId:))`.
   - `.navigationDestination(for: HomeDestination.self)` → for now a placeholder
     `Text(deckId)` view marked `// TODO(T045): replace with DeckDetailView`.
   - Bottom content inset ~90pt so the floating tab bar never covers the last row.
4. Wire into the shell: in `Sources/App/MainShellView.swift`, replace the Learn tab's
   placeholder with `HomeView()`. Touch nothing else in the shell.
5. Tests in `Tests/HomeViewModelTests.swift` (XCTest, in-memory `PersistenceStack.container`,
   real bundled catalog via `ContentStore`): use a fixed `DateProvider` stub.
   - Subtitle math: trip started 1 day before `now` with countryId "japan", duration
     "weekend" → subtitle == "Weekend in Tokyo · Day 2".
   - Continue-deck selection: two owned decks, first fully learned, second partially → picks
     the second; all owned decks untouched → picks the first owned (status new).
   - Estimate string: 22 cards remaining → "about 9 min left" (22×25s = 550s → 9.17 → 9).
   - Banner visibility: `dueCount == 0` ⇒ no banner state; > 0 ⇒ banner state present.
