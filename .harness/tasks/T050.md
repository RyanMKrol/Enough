# T050 ‚Äî Session complete screen (ring race, check pop, stats, learn-5-more)

## Overview

Build the session-complete screen ‚Äî ring racing to full, check pop, three stat tiles, and the
Done / "Learn 5 more" actions ‚Äî and swap it in for the placeholder `SessionRoute.complete`
destination both session screens navigate to.

## Do

Design reference: .harness/docs/designs/design-spec.md (read-only reference) ¬ß6c. Uses `ProgressRing`,
`CheckPopView`, `PrimaryButtonStyle`, `TextLinkButtonStyle`, `StatsService`,
`DeckProgressService`, `StudyService`; accent via `\.accentTheme`.

1. `Sources/Features/Session/SessionCompleteView.swift` ‚Äî `struct SessionCompleteView: View`
   with `init(summary: SessionSummary, deckId: String?, onDone: @escaping () -> Void)`
   (`deckId` = the session's deck for learn/practice; nil for review sessions). Centered
   VStack on white:
   - **Hero** (ZStack, ~130pt): `ProgressRing(progress: 1.0, size: 120, lineWidth: 8, tint:
     accentTheme.accent, showsPercent: false)` ‚Äî its built-in fill animation "races" to full
     on appear; overlaid `CheckPopView(size: 44, color: accentTheme.accent)` delayed 0.3s
     (`.task { try? await Task.sleep(for: .seconds(0.3)); showCheck = true }`).
   - Title "Round complete" ‚Äî 30 Bold, top padding 18.
   - Subline 15 Regular `#6b6b70`, centered, max-width 300:
     - review mode: "You cleared today's reviews. \(strongestDeckTitle) is holding strong."
       ‚Äî strongest = OWNED deck with the highest `DeckProgressService.strength`; tie-break:
       most `mastered`; resolve titles via `ContentStore`.
     - learn/practice mode: "That's \(deckTitle) under way. \(summary.cardsCompleted) cards
       down."
   - **Three stat tiles** (HStack, spacing 10; each: white card radius 18, hairline border,
     padding 14, equal widths): big value 22 Bold + caption 12 Regular `#8e8e93`:
     "\(summary.cardsCompleted)" / "cards"; formatted `summary.duration` as m:ss (e.g.
     "4:10") / "minutes"; "\(streak)" / "day streak" ‚Äî streak read from
     `StatsService.currentStreak()` AT APPEAR TIME (the session was already committed before
     navigation, so today's activity is included and a streak tick shows).
   - "Done" ‚Äî `PrimaryButtonStyle(background: accentTheme.accent)` ‚Üí `onDone()` (dismisses
     the full-screen session cover back to the launching context).
   - "Learn 5 more" ‚Äî `TextLinkButtonStyle` ‚Äî shown ONLY if
     `services.study.makeLearnMoreSession(size: 5)` can produce a non-empty engine (new cards
     remain among owned decks); tapping replaces the current session content with
     `MCSessionView(engine:)` for that engine.
   - Motion discipline: the ring fills once, the check pops once ‚Äî nothing loops, no
     confetti (design: "one tasteful beat, then out").
2. Replace the placeholder: in BOTH `MCSessionView` and `ReviewSessionView`, the
   `.navigationDestination(for: SessionRoute.self)` now returns `SessionCompleteView(
   summary: engine.summary(), deckId:, onDone:)` instead of the T047 `Text` placeholder ‚Äî
   define the destination once in a small shared helper in
   `Sources/Features/Session/SessionRoute.swift` if that avoids duplication.
3. `#Preview`s: review variant (12 cards, 4:10, streak 7) and learn variant.

Copy strings EXACT: "Round complete", "Done", "Learn 5 more",
"You cleared today's reviews. \(deck) is holding strong.",
"That's \(deck) under way. \(n) cards down.".
