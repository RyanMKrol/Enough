# T004 — Chips & labels: scenario chip, meta chip, eyebrow label

## Overview

Add three small reusable text components: the multi-select scenario chip (onboarding Step 2),
the translucent meta chip (tinted hero headers), and the uppercase eyebrow label (step markers,
card section headers). Design values from design-spec.md §1.2/§2.3/§5.

## Do

1. Create `Sources/DesignSystem/Chips.swift` with exactly these three views:
   - `struct ScenarioChip: View` with `init(title: String, isSelected: Bool,
     action: @escaping () -> Void)`. A `Button` rendering a `Capsule` chip: text 15pt semibold,
     padding 10 vertical / 16 horizontal. Reads `@Environment(\.accentTheme) var accentTheme`.
     Selected: fill `accentTheme.tint`, text `accentTheme.accent`, and a `Capsule` stroke overlay
     in `accentTheme.accent` at `lineWidth: 1.5` (inset so it doesn't clip). Unselected: fill
     `EnoughColor.insetSurface`-adjacent `#f2f2f7` (use `EnoughColor.canvas`), text
     `EnoughColor.secondaryText` (`#6b6b70`), no ring. State changes animate with
     `.spring(response: 0.3, dampingFraction: 0.6)` keyed on `isSelected` (the "fills with a
     small spring" behavior, design-spec §2.3).
   - `struct MetaChip: View` with `init(text: String)`. A static (non-interactive) `Capsule`
     pill for use on accent-tinted hero headers: fill `Color.white.opacity(0.25)`, text white
     13pt semibold, padding 5 vertical / 10 horizontal.
   - `struct EyebrowLabel: View` with `init(_ text: String)`. Renders `text` uppercased, font
     `EnoughFont.eyebrow()` (12 semibold), kerning `12 * EnoughFont.eyebrowTracking` (0.06em),
     color `EnoughColor.tertiaryText` (`#8e8e93`).
2. Add a `#Preview` showing: a selected + an unselected ScenarioChip ("Eating out",
   "Nightlife"), a row of MetaChips ("30 cards", "~12 min", "Audio") over an accent-colored
   rectangle, and an EyebrowLabel ("Step 1 of 3").

Conventions: single app target, SwiftUI only, 2-space indent, SwiftLint strict (≤130 cols) +
swift-format strict. No business logic — selection state is owned by the caller. Touch only
`Sources/DesignSystem/**` (and `Tests/**` if adding optional tests).

## Done when

- `Sources/DesignSystem/Chips.swift` compiles with the exact three view names and initializer
  signatures above.
- ScenarioChip's two states match the pinned values (selected: accent tint fill + 1.5pt accent
  ring + accent text; unselected: `#f2f2f7` fill + `#6b6b70` text) and re-tint via the
  `\.accentTheme` environment.
- EyebrowLabel uppercases its input and uses 12pt semibold with positive letter tracking;
  MetaChip is white-on-translucent-white as specified.
- The `#Preview` block exists and renders all three components as described.
