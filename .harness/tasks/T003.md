# T003 — Button styles: primary (graphite/accent), tinted secondary, text link

## Overview

Add the three reusable button styles every screen uses: the filled primary button (graphite by
default, accent-overridable — onboarding animates that color), the accent-tinted secondary
button, and the blue text-link button. Pure presentation; design values from design-spec.md §1/§2.

## Do

1. Create `Sources/DesignSystem/Buttons.swift` containing exactly these three `ButtonStyle`
   types (names are load-bearing — later tasks reference them verbatim):
   - `struct PrimaryButtonStyle: ButtonStyle` with `init(background: Color = EnoughColor.graphite,
     subtleShadow: Bool = false)`.
     Label: white, 17pt semibold, centered; frame `maxWidth: .infinity`, height
     `Layout.buttonHeight` (54); background = the `background` color in a
     `RoundedRectangle(cornerRadius: Layout.buttonRadius)` (16). Shadow: color
     `background.opacity(0.35)`, radius 9, y 6 — but when `subtleShadow` is true use radius 5,
     y 3, opacity 0.22 (used where a secondary button sits directly beneath, per design-spec §5).
     Pressed state: `scaleEffect(0.98)` + opacity 0.9, animated `.easeOut(duration: 0.15)`.
     IMPORTANT: the fill must be driven by the `background` parameter so a caller changing it
     (graphite → accent morph, design-spec §2) can animate the color with `withAnimation` — do
     not hardcode graphite inside `makeBody`.
   - `struct TintedButtonStyle: ButtonStyle` with `init()`. Reads
     `@Environment(\.accentTheme) var accentTheme` inside the style; fill = `accentTheme.tint`,
     text = `accentTheme.deep`, 17pt semibold; same 54pt height / radius-16 shape; no shadow;
     pressed = scale 0.98.
   - `struct TextLinkButtonStyle: ButtonStyle` with `init()`. Text-only: `EnoughColor.linkBlue`,
     15pt regular, no background; pressed = opacity 0.6.
2. Add a `#Preview` in the same file showing all three stacked: a graphite primary
   ("Get started"), an accent primary via `PrimaryButtonStyle(background:
   AccentTheme.japan.accent)` ("Continue"), a tinted secondary ("Practice all 30"), and a text
   link ("Restore purchases").

Conventions: single app target (no imports beyond SwiftUI), 2-space indent, SwiftLint strict
(lines ≤ 130) + swift-format strict. No business logic. Touch only `Sources/DesignSystem/**`
(and `Tests/**` if you add optional tests).

## Done when

- `Sources/DesignSystem/Buttons.swift` compiles with the exact three type names and initializer
  signatures above; `PrimaryButtonStyle(background:)` renders the passed color and
  `PrimaryButtonStyle()` defaults to graphite `#1c1c1e`.
- Buttons are 54pt tall, radius 16, full-width; primary text is white 17 semibold; tinted style
  pulls its colors from the `\.accentTheme` environment (so the same button re-tints when the
  environment changes); text link is `#3d7ede` at 15pt.
- The `#Preview` block exists and renders the four variants described above.
