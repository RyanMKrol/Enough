## Overview

Bundle the repo's `Content/` fixture (catalog.json, decks/, audio/) into the app as a folder
reference and add `ContentStore`, the single read-only gateway every feature uses to load the
catalog, a deck's cards, and per-card audio URLs. The models it returns were defined in T011
(`Sources/ContentKit/Models.swift`).

## Do

1. Edit `project.yml`: give the `Enough` target a resources build-phase entry for the `Content`
   directory as a FOLDER reference (so the bundle contains a `Content/` subdirectory preserving
   the catalog JSON, deck JSONs, and audio mp3s under the repo-root Content directory — read-only data, never edited by this task). Exact YAML — add this
   `resources` key alongside the target's existing `sources`:

   ```yaml
   targets:
     Enough:
       # ... existing keys unchanged ...
       sources:
         - Sources
         - path: Content
           type: folder
           buildPhase: resources
   ```

   (XcodeGen: a source entry with `type: folder` + `buildPhase: resources` produces a blue
   folder reference copied into the bundle.) Change nothing else in `project.yml`.

2. Create `Sources/ContentKit/ContentStore.swift`:

   ```swift
   enum ContentStoreError: Error { case missingResource(String), deckNotFound(String), countryNotFound(String) }
   final class ContentStore {
     init(bundle: Bundle = .main)
     func catalog() throws -> ContentCatalog          // decodes Content/catalog.json, cached after first load
     func cards(forDeck deckId: String) throws -> [CardContent]  // via the deck's cardsFile, cached per deck
     func audioURL(forFile name: String) -> URL?      // URL of Content/audio/<name>, nil if absent
     func country(_ id: String) throws -> CountryInfo
     func deck(_ id: String) throws -> DeckInfo       // searches every country's decks
   }
   ```

   Resource lookup semantics for a folder reference: `bundle.url(forResource: "catalog",
   withExtension: "json", subdirectory: "Content")`; deck files via subdirectory
   `"Content/decks"`; audio via subdirectory `"Content/audio"` (split the file name into
   resource + extension). `cards(forDeck:)` resolves the deck's `cardsFile` (e.g.
   `"decks/jp-greetings.json"`) relative to `Content/`. Throw `missingResource` when a URL is
   nil, `deckNotFound`/`countryNotFound` for unknown ids. Cache the decoded catalog and each
   decoded deck in private properties so repeat calls don't re-read the bundle.

3. Treat `Content/**` as READ-ONLY truth: do not edit, rename, or "fix" any fixture file. If a
   test expectation seems to mismatch the fixture, the test expectation is wrong — re-check the
   fixture values below.

4. Create `Tests/ContentStoreTests.swift` (XCTest, `ContentStoreTests`). The unit-test bundle is
   HOSTED by Enough.app, so in tests `Bundle.main` IS the app bundle — construct the store as
   `ContentStore()` (default `.main`).
