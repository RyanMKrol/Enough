## Overview

Add `EntitlementStore`, the persistence gateway for what the user owns (individual decks and
bundles), including expanding an owned bundle into its member deck ids via the content catalog.
Builds on `EntitlementRecord` from T014 (do not change its fields) and the ContentKit models
from T011 (`ContentCatalog`, `BundleInfo` — same app target, no import needed).

## Do

1. Create `Sources/Persistence/EntitlementStore.swift`:

   ```swift
   final class EntitlementStore {
     init(context: ModelContext)

     /// Inserts an EntitlementRecord(productId:kind:grantedAt:) and saves.
     /// IDEMPOTENT: if a record with the same productId already exists, does nothing.
     func grant(productId: String, kind: String, now: Date) throws   // kind: "deck" | "bundle"

     /// All granted productIds (decks and bundles, unexpanded).
     func grantedProductIds() throws -> Set<String>

     /// Every OWNED deck id: directly-granted decks, plus the deckIds of every granted
     /// bundle, resolved against the catalog.
     func ownedDeckIds(catalog: ContentCatalog) throws -> Set<String>

     func isOwned(deckId: String, catalog: ContentCatalog) throws -> Bool

     /// Deletes all EntitlementRecord rows and saves.
     func reset() throws
   }
   ```

   - `ownedDeckIds` iterates `catalog.countries` → `bundles`; for each bundle whose `id` is in
     `grantedProductIds()`, union its `deckIds`. Then union directly-granted ids that match any
     deck id in the catalog. Granted ids that match nothing in the catalog are ignored (stale
     grants must not crash).
   - Use plain strings for `kind` ("deck"/"bundle") to match `EntitlementRecord.kind`.

2. Create `Tests/EntitlementStoreTests.swift` (XCTest, class `EntitlementStoreTests`). Every
   test uses `try PersistenceStack.container(inMemory: true)` + a fresh `ModelContext`. Build a
   small `ContentCatalog` fixture IN CODE (not from the bundle): one country with deck ids
   `["d1", "d2", "d3"]` and one bundle `"b1"` whose `deckIds == ["d1", "d2"]` (construct the
   T011 structs directly with dummy strings for the other fields).

Conventions: single app target (`@testable import Enough`), Swift 5 / iOS 18, 2-space indent,
SwiftLint strict (lines ≤ 130).

## Done when

- `Tests/EntitlementStoreTests.swift` asserts:
  - Empty store: `grantedProductIds()` empty; `ownedDeckIds(catalog:)` empty;
    `isOwned(deckId: "d1", ...)` false.
  - After `grant(productId: "d3", kind: "deck", ...)`: `ownedDeckIds` == `["d3"]`;
    `isOwned("d3")` true; `isOwned("d1")` false.
  - After additionally `grant(productId: "b1", kind: "bundle", ...)`: `ownedDeckIds` ==
    `["d1", "d2", "d3"]` — the bundle expanded to its member decks.
  - Idempotency: calling `grant(productId: "b1", kind: "bundle", ...)` twice leaves exactly 2
    `EntitlementRecord` rows total (fetch and count).
  - A granted id that appears nowhere in the catalog (e.g. `"ghost"`) leaves `ownedDeckIds`
    unchanged and does not throw.
  - After `reset()`: `grantedProductIds()` is empty.
- No file outside `Sources/Persistence/` and `Tests/` changes.
