# T009 — Welcome card-stack hero (premium panel)

## 2026-07-15

### What was done

Built the welcome screen's hero component: a three-card fanned stack with gentle bobbing animation, the front card showing the premium panel composition (Japan header band, ありがとう text, romaji pronunciation pill).

**Implementation details:**
- Created `Sources/DesignSystem/CardStackHero.swift` with `struct CardStackHero: View` and parameterless `init()`
- Fixed stage frame: **255 × 292**
- Three cards with `RoundedRectangle(cornerRadius: 28)`, each **165 × 224**:
  - Back-left: `#e8f5ff` fill, −10° rotation, offset (−36, 6)
  - Back-right: `#fff0df` fill, +10° rotation, offset (36, 6)
  - Front: white fill with 0.07-alpha border and 0.14-alpha shadow (radius 21, y 20)
- Front card content, clipped to card shape:
  - Header band: `#fff0ee` background with `#fedbd7` 1pt bottom border
    - HStack with 30×30 white rounded-rect chip containing 13pt `#e24947` circle
    - "Japan" text 13pt semibold `#a43b38`
  - Stage below: centered `Text("ありがとう")` 24pt bold, pronunciation pill (`Text("arigatō")` 12.5pt medium `#a45953` on `#ffe9e6`, fully rounded, kerning 12.5 × 0.04)
- Bob animation using Motion.bobDuration (4s):
  - Front card: ±6pt, phase 0 (starts immediately)
  - Back-left card: ±4pt, phase +0.5s
  - Back-right card: ±4pt, phase +1.0s
  - All ease-in-out, repeat forever with autoreverse
- Added `#Preview` rendering `CardStackHero()` on white background
