# T003 — Button styles: primary (graphite/accent), tinted secondary, text link

## 2026-07-15

**Status:** done

**What was done:**
- Created `Sources/DesignSystem/Buttons.swift` with three ButtonStyle implementations:
  1. `PrimaryButtonStyle` with parameterized background color (defaults to graphite #1c1c1e), supports subtle shadow mode, 54pt height, radius 16, white 17pt semibold text, shadow with opacity/radius variants, scale 0.98 + opacity 0.9 on press (animated .easeOut 0.15s)
  2. `TintedButtonStyle` reads `@Environment(\.accentTheme)`, uses tint color for background and deep color for text, same dimensions and press behavior
  3. `TextLinkButtonStyle` text-only style with linkBlue (#3d7ede), 15pt regular, opacity 0.6 on press
- Added #Preview block showing all four variants: graphite primary, accent primary (Japan red), tinted secondary, and text link
