# T025 — Worklog

## 2026-07-16 — Implementation complete, all tests passing

**What was built:**
- `Sources/Services/PricingCalculator.swift` — pure pricing calculations enum with 4 static methods:
  - `price(_:)` — formats GBP amounts to localized strings ("£2.49") using a cached NumberFormatter
  - `bundleSavings(_:in:)` — calculates savings by comparing individual deck prices to bundle price, rounded to 2 dp
  - `planTotal(selectedBundle:extraDeckIds:country:)` — computes total cost of selected bundle plus any extra decks not already in the bundle, rounded to 2 dp
  - `summaryLine(selectedBundle:extraDeckIds:)` — generates human-readable summary ("Weekend · 3 packs" or "1 pack" or "No packs")
- `Tests/PricingCalculatorTests.swift` — 10 XCTest tests covering:
  - GBP formatting (2.49 → "£2.49", 1 → "£1.00", 8.99 → "£8.99")
  - Bundle savings calculations (jp-weekend and jp-week both 0.51)
  - Plan totals with bundles + extras, excluding already-bundled decks
  - Plan totals without bundles (à-la-carte)
  - Summary line formatting in all modes (bundle+count, extras, singular/plural)
