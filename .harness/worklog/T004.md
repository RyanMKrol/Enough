# T004 — Chips & labels: scenario chip, meta chip, eyebrow label

## 2026-07-15 21:53

**Outcome:** ✅ done

### What was done

Created `Sources/DesignSystem/Chips.swift` with three components:

1. **ScenarioChip** — A multi-select chip button with:
   - Init: `(title: String, isSelected: Bool, action: () -> Void)`
   - Selected state: accent-tinted fill + 1.5pt accent ring + accent text
   - Unselected state: `#f2f2f7` canvas fill + `#6b6b70` secondary text
   - Spring animation on state change (response: 0.3, dampingFraction: 0.6)
   - Reads accent theme from environment
   - Text: 15pt semibold with 10 vertical / 16 horizontal padding

2. **MetaChip** — A static translucent pill for hero headers:
   - Init: `(text: String)`
   - Fill: `Color.white.opacity(0.25)`
   - Text: white, 13pt semibold, 5 vertical / 10 horizontal padding
   - Capsule shape

3. **EyebrowLabel** — An uppercase decorative label:
   - Init: `(_ text: String)`
   - Renders text uppercased
   - Font: 12pt semibold (EnoughFont.eyebrow)
   - Kerning: 12 * 0.06 (0.72 standard units for 12em tracking)
   - Color: `#8e8e93` (tertiaryText)
