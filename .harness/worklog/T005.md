# T005 — Progress indicators: animated progress bar + progress ring

## Built 2026-07-15

### Summary

Implemented two progress indicator components in `Sources/DesignSystem/ProgressViews.swift`:
- **AnimatedProgressBar**: thin capsule track with animated fill (0→progress) over 1.0s easeOut
- **ProgressRing**: circular progress ring with optional percent label, animates fill over 1.2s easeOut

Both components:
- Clamp progress to 0…1 range (no crashes on out-of-range values)
- Animate on first appearance and on all subsequent value changes
- Use Motion.barGrow (1.0s) and Motion.ringFill (1.2s) durations respectively
- Support all exact signatures specified in task

### Design implementation notes

- AnimatedProgressBar: Uses GeometryReader to calculate width from progress × available width. Leading-aligned capsule fill with scaleX behavior via animation of displayedProgress.
- ProgressRing: Circle().trim() with rotation −90° to start at 12 o'clock. Percent label centered using font size * 0.28 (matches spec ratio). Round line caps for polish.
- Preview: Shows bar at 0.27 (Japan accent), ring at 0.4 (with %), ring at 1.0 (without %).
