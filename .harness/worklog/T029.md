# T029 — AppServices, AppState & RootView (app shell wiring)

## 2026-07-16

**Built:** the app's dependency container, top-level observable state, and root view, and
rewired `EnoughApp` to use them.

**Implementation:**
- `Sources/App/AppServices.swift` — `final class AppServices` holding `dateProvider`,
  `contentStore`, `container`, `tripStore`, `entitlementStore`, `cardSRSStore`,
  `activityStore`, `purchase`, `audio`, `study`, `deckProgress`, `stats` as `let` properties;
  a designated `init` taking every dependency; `static func live()` (on-disk
  `PersistenceStack.container()`, `AdjustableDateProvider`, `StubPurchaseService`) and
  `static func preview()` (in-memory container, same graph) via a shared private
  `makeGraph(container:)` helper. Also adds `EnvironmentValues.services` via `@Entry`,
  defaulting to `.preview()`.
- `Sources/App/AppState.swift` — `@Observable final class AppState` with `Phase` (
  `.onboarding`/`.main`), `activeAccent: AccentTheme`. `init(services:)` reads
  `services.tripStore.activeTrip()`: non-nil active trip → `.main` +
  `AccentTheme(rawValue: trip.accentRawValue) ?? .japan`; nil → `.onboarding` + `.japan`.
  `completeOnboarding(accent:)` sets accent and flips to `.main`.
  `startNewTrip()` flips to `.onboarding` only — entitlements/SRS data untouched.
- `Sources/App/RootView.swift` — switches on `appState.phase`, showing `Text("Onboarding")`
  or `Text("Main")` placeholders (replaced by T031/T030 respectively), applying
  `.environment(\.accentTheme, appState.activeAccent)`.
- Rewired `Sources/EnoughApp.swift`: builds `services`/`appState` once via `@State` +
  custom `init()`, shows `RootView(appState: appState)`, applies
  `.environment(\.services, services)` and `.modelContainer(services.container)`.
- Deleted `Sources/ContentView.swift` (T001 placeholder) — nothing references it anymore.

**Tests written (`Tests/AppStateTests.swift`, XCTest, in-memory via `AppServices.preview()`):**
- empty trip store ⇒ `.onboarding` + `.japan`.
- active trip with `accentRawValue: "france"` ⇒ freshly constructed `AppState` is `.main` +
  `.france`.
- `startNewTrip()` flips `.main` → `.onboarding` while a previously granted entitlement
  remains owned.
