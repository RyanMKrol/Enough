# T038: Debug setting: unlock all packs

## 2026-07-16 · 11:25

### What was done

Implemented the debug-menu action to unlock all packs in the catalog:

1. **DebugEntitlementTools.swift** — New enum with `grantAll()` static method that grants every deck and bundle in the catalog via `EntitlementStore.grant()`. Idempotent: granting an already-owned product returns early without error. Returns the count of all decks in the catalog.

2. **DebugUnlock.swift** — New enum with a `DebugSection` containing one `.action` row that loads the catalog, calls `grantAll()`, and returns a toast message "Unlocked N packs".

3. **DebugRegistry.swift** — Updated `DebugProviders.all` to include `DebugUnlock.section` alongside the existing `DebugAbout.section`.

4. **DebugEntitlementToolsTests.swift** — New test file with one test case (`testGrantAllGrantsAllDecksAndBundlesAndIsIdempotent`) that:
   - Creates an in-memory `ModelContainer` and `EntitlementStore`
   - Loads the real bundled catalog via `ContentStore`
   - Calls `grantAll()` twice
   - Asserts that `ownedDeckIds()` equals all deck IDs across all countries
   - Verifies idempotence: second call returns the same count and doesn't throw
