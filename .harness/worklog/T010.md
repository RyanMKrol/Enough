# T010 — SRS grade button row

**Date:** 2026-07-15

## What I did

Created `Sources/DesignSystem/GradeButtons.swift` with:

1. **`GradeChoice` enum** — A CaseIterable enum with four cases (again, hard, good, easy) and computed properties:
   - `title` — Returns "Again", "Hard", "Good", "Easy"
   - `fill` — Returns color-coded fills:
     - again: `#ffe9e6` (red-tint)
     - hard: `#f2f2f7` (neutral canvas)
     - good: `EnoughColor.successTint` (`#d8f9dd`)
     - easy: `EnoughColor.easyTint` (`#caf7ff`)
   - `textColor` — Returns color-coded text colors:
     - again: `#b71824` (dark red)
     - hard: `EnoughColor.secondaryText` (`#6b6b70`)
     - good: `EnoughColor.successDeep` (`#00722e`)
     - easy: `EnoughColor.easyBlue` (`#1666aa`)

2. **`GradeButtonRow` view** — A SwiftUI View that:
   - Takes `previews: [GradeChoice: String]` (interval sublabels) and `onGrade: @escaping (GradeChoice) -> Void` callback
   - Renders an HStack of 4 equal-width buttons (one per GradeChoice.allCases in declaration order)
   - Each button displays a VStack with:
     - Title (15pt semibold, color-coded)
     - Interval sublabel from previews dict (11pt regular, 70% opacity text color)
   - Each button has ~56pt height with `RoundedRectangle(cornerRadius: Layout.chipRadius)` background
   - Tap calls `onGrade(choice)`
   - Pressed state scales to 0.97

3. **`GradeButtonStyle` — A ButtonStyle** that handles the scale animation for press state

4. **Preview** — Shows the component with example intervals ("<1 min", "1 day", "3 days", "6 days") on a canvas background
